# EduMap Frontend (Web)
Dự án này sử dụng Next.js 14 (App Router) + TypeScript.

## Yêu cầu cài đặt
- Node.js >= 18
- npm hoặc yarn

## Cấu trúc thư mục dự kiến
- `app/`: Các trang (Pages) & API routes
- `components/`: UI Components (Button, Card, MapViewer)
- `hooks/`: Custom React Hooks
- `services/`: Kết nối API đến Backend (NestJS)
- `styles/`: CSS & Tailwind config
